# Privacy Policy

Last updated: April 19, 2026

## Overview

This application is a closed test of a WhatsApp-based messaging workflow.

Its current purpose is:

- testing message delivery and connection behavior between WhatsApp, Meta, and OpenAI.
- testing the use of OpenAI-based AI to help generate message responses
- evaluating the behavior of the system in a limited, informed test group

This is not currently offered as an open public service. It is being tested with a closed group of people who are informed that the system is under evaluation.

## What data we process

When a participant sends a WhatsApp message to the test number, we may process:

- the sender's phone number
- the sender's WhatsApp profile information made available through the platform
- the content of messages sent to the test number
- message metadata, such as timestamps and delivery-related event data
- technical logs needed to test connectivity, workflow execution, and response behavior

## How the data is processed

Messages are received through Meta's WhatsApp Cloud API and webhook system.

They are processed by a self-hosted software instance running on a local macOS computer for workflow testing.

Message content may also be sent to OpenAI services for the purpose of testing AI-assisted replies.

## Why we process the data

We process this data only for the following purposes:

- testing the technical connection between WhatsApp, Meta, and OpenAI
- testing automated reply flows
- testing and evaluating AI-assisted response generation
- diagnosing errors, webhook behavior, and message-handling issues
- improving the test workflow during the closed evaluation period

## Legal/operational basis of the test

The system is currently used only in a closed testing environment with informed participants.

Participants are informed that:

- the system is under test
- responses may be automated
- AI may be used to help generate replies
- message handling may be reviewed for testing and debugging purposes

## Data retention

Data is retained only as long as reasonably necessary for testing, debugging, and evaluating the system.

Because this is an evolving test environment, retention periods may vary depending on technical needs, incident analysis, and workflow adjustments.

If a participant requests deletion, we will make a good-faith effort to delete or anonymize stored data associated with that participant, subject to any legal or operational obligations that require temporary retention.

## Data sharing

We do not sell participant data.

Data may be processed by service providers involved in the test workflow, including:

- Meta, for WhatsApp Cloud API message transport and webhook delivery
- OpenAI, for AI-assisted response testing
- infrastructure or hosting providers used to publish the legal information pages
